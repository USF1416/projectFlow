ProjectFlow is a simplified Jira-like web app built for portfolio purposes (BTS SIO SLAM).
It focuses on clean UI, reusable components, and basic issue management workflows.

## Features
- Modern SaaS UI (light mode, blue theme)
- Dashboard with statistics and recent issues
- Projects list with progress (computed from issues)
- Global Kanban board (To Do / In Progress / Done)
- Project filter on the board
- Issue modal: edit title, status, priority, assignee, description (instant UI update)

## Tech Stack
- React (Vite)
- React Router
- Plain CSS (no Tailwind)
- Local mock data (no backend)
